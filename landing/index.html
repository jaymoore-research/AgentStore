Install agent skills without wrestling GitHub.

AgentStore installs and manages AI agent packages locally. Point it at a GitHub repo, it clones, detects components, and symlinks skills into your tools. No server, no account, no telemetry.

Prefer the command line?

A single 3 MB binary. No runtime dependencies. Build from source or download from releases.

# Install a package from GitHub
agentstore install anthropics/skills --platform claude,cursor

# See what's installed
agentstore list

# Update a package
agentstore update skills

# Check detected platforms
agentstore platforms

# Build from source
cargo build --release -p agentstore-cli

How it works

Give it a repo. It handles the rest.

01

Point at a repo

Enter an owner/repo (GUI) or run agentstore install owner/repo (CLI). The repo is cloned locally with a shallow depth-1 fetch.

02

Auto-detect components

AgentStore scans the clone for skills, instructions, MCP servers, hooks, and keybindings. It figures out which platforms the package supports.

03

Symlink into your tools

One copy on disk, symlinked into Claude Code, Cursor, Copilot, Codex, and OpenCode. Toggle platforms on and off. Updates re-clone and re-link.
